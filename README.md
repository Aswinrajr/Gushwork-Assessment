# Meera Industries - Premium HDPE Pipes & Coils Landing Page

A high-performance, modern, and fully responsive product landing page built for **Meera Industries**, showcasing their premium HDPE (High-Density Polyethylene) pipes and industrial machinery solutions. This project was developed as part of a technical assessment, focusing on premium UI/UX, smooth interactions, and robust frontend implementation.

## 🚀 Live Demo

You can view the live deployment of this project here:  
**[gushwork-assessment.vercel.app](https://gushwork-assessment.vercel.app/)**  
_(Deployment via Vercel)_

## ✨ Key Features

### 1. Interactive Product Showcase

- **Dynamic Image Carousel**: Users can navigate through product images using thumbnails, navigation arrows, or by directly clicking the main image.
- **Magnifier Zoom Effect**: Precise hover-to-zoom functionality on the main product image for detailed inspection of technical features.

### 2. Advanced Navigation & UI Componentry

- **Smart Sticky Header**: A sleek, scroll-aware header that hides on scroll down to maximize screen real estate and reappears on scroll up for quick navigation.
- **Responsive Mobile Menu**: A dedicated mobile side-drawer with glassmorphism (backdrop-blur) effects, ensuring a premium feel on smaller devices.
- **Animated Marquee**: A seamlessly looping "Trusted Companies" marquee that pauses on hover, providing subtle yet effective social proof.

### 3. Rich Content Sections

- **Manufacturing Process Tabs**: An interactive tabbed interface that walks users through the 8-step production process, complete with synchronized titles and visuals.
- **Applications Slider**: A horizontal, touch-friendly slider highlighting various industry use-cases.
- **Technical Specifications**: A comprehensive, well-structured table detailing industrial parameters, optimized for readability across all devices.
- **FAQ Accordion**: A clean, animated accordion for frequently asked questions, helping users find information quickly.

### 4. Conversion & Lead Generation

- **Integrated Modals**: Custom-built modal systems for "Catalogue Requests" and "Expert Consultations," triggered by various CTAs throughout the page.
- **Interactive Forms**: Validation-ready forms for quote requests and expert call-backs.
- **Resources & Downloads**: Dedicated section for accessing technical manuals and engineering data sheets.

## 🛠️ Technology Stack

- **Structure**: Semantic HTML5 for better SEO and accessibility.
- **Styling**: Vanilla CSS3 with:
  - **CSS Variables**: For consistent theming and easy maintenance.
  - **Modern Layouts**: Robust use of CSS Grid and Flexbox for responsiveness.
  - **Custom Animations**: Smooth transitions and keyframe animations for marquee and slider effects.
  - **Fluid Typography**: Leveraging `clamp()` for perfect text sizing across all resolutions.
- **Logic**: Vanilla JavaScript (ES6+) — No external libraries (like jQuery) were used, ensuring lightning-fast performance and zero dependency overhead.
- **Typography**: Google Fonts (Inter) for a clean, professional aesthetic.
- **Icons**: Optimized SVG icons for crisp visuals on all displays.
